# CogKGE Model

Download CogKGE Model from http://210.75.240.138:8080/download/CogNLP_data/CogKTR_data/KNOWLEDGE_COGKGE.zip  
Put the raw data in path`datapath/knowledge_graph/cogkge/raw_data`.   
Like the following form:

```angular2html
datapath
├─ knowledge_graph
│  ├─ cogkge
│  │  ├─ raw_data
│  │  │  ├─ Model.pkl
│  │  │  └─ vocab.pkl
│  │  │
```
