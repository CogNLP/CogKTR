# S20Rel

Download S20Rel from https://www.dropbox.com/s/s8zb8o5agpgx1e9/data_model.zip?dl=0
Put the raw data in path`datapath/knowledge_graph/S20Rel`.  
Like the following form:

```angular2html
datapath
├─ knowledge_graph
│  ├─ S20Rel
│  │  ├─ entity2id.txt
│  │  ├─ partition_20.txt
│  │  ├─ relation2id.txt
│  │  ├─ train2id.txt
```
